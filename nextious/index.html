Nextious - 局域网通信与文件传输解决方案

基于局域网的即时通信和文件传输系统，提供安全、快速、便捷的本地网络通信体验

了解更多 查看文档

核心特性

即时通信

支持文字、表情、图片等多种消息类型，实时同步，沟通无障碍

文件传输

高速文件传输，支持断点续传，大文件传输更稳定

安全加密

端到端加密通信，确保数据传输安全，保护用户隐私

技术架构

客户端

  • • React + TypeScript 构建现代化用户界面
  • • WebSocket 实时通信协议
  • • 响应式设计，支持多设备访问

服务端

  • • Node.js + Express 高性能后端服务
  • • Socket.io 实时双向通信
  • • SQLite 轻量级数据存储

快速开始

1

下载安装

从GitHub Releases下载最新版本，支持Windows、macOS、Linux

2

启动服务

运行服务端程序，自动发现局域网内的其他用户

3

开始通信

打开客户端，连接本地服务器，即可开始聊天和文件传输
